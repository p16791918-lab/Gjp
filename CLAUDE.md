# Claude Code 작업 플로우

교시별 분석은 반드시 아래 섹션 순서대로 진행하며, **각 섹션 완료 후 반드시 멈추고 사용자의 확인을 받은 뒤** 다음 섹션으로 넘어간다. 여러 교시를 한 번에 처리하지 않는다.

---

## 교시별 분석 플로우 (1교시 → 2교시 → 3교시 순서로 각각 진행)

### 섹션 A: 지침 읽기

**작업 내용:**
- 해당 교시의 정답지(XLSX) 파일을 읽어 과목 구성과 문항 수를 파악한다
- 기존 `make_pdf_*.py` 스크립트가 있다면 현재 구조를 확인한다
- 분석 대상 연도(2021, 2023, 2024, 2025)의 파일 목록을 확인한다

**완료 기준:** 해당 교시 파일 구조 및 과목 구성 파악 완료

> **멈춤 포인트:** 섹션 A 완료. 파악한 파일 목록과 과목 구성을 보고하고, 섹션 B로 넘어갈지 확인을 받는다.

---

### 섹션 B: 문제 읽기

**작업 내용:**
- 해당 교시의 각 연도별 문제 PDF를 순서대로 읽는다 (한 번에 한 연도씩)
- 각 문항의 주제, 핵심 키워드, 출제 패턴을 기록한다
- 연도 간 반복 출제 항목을 표시한다

**완료 기준:** 4개 연도 문제 전체 검토 및 반복 출제 항목 목록 작성 완료

> **멈춤 포인트:** 섹션 B 완료. 연도별 문제 검토 결과와 반복 출제 항목 목록을 보고하고, 섹션 C로 넘어갈지 확인을 받는다.

---

### 섹션 C: 빈출 유형 분석

**작업 내용:**
- 3~4년 연속 출제: ★★★ 최빈출 유형으로 분류
- 2년 출제: ★★ 준빈출 유형으로 분류
- 각 유형별 핵심 포인트 및 오답 선택지 패턴을 정리한다
- 주제별 핵심 요약을 작성한다

**완료 기준:** 최빈출/준빈출 유형 목록 및 핵심 포인트 정리 완료

> **멈춤 포인트:** 섹션 C 완료. 분류된 빈출 유형 목록을 보고하고, 섹션 D(스크립트 작성)로 넘어갈지 확인을 받는다.

---

### 섹션 D: PDF 생성 스크립트 작성

스크립트를 한 번에 작성하면 오류가 발생하므로 **5단계로 나눠서** 진행한다.  
각 단계 완료 후 반드시 멈추고 확인을 받은 뒤 다음 단계로 넘어간다.

#### D-1: 골격 + 표지 + ★★★ 최빈출 첫 번째 과목

**작업 내용:**
- import, 폰트 등록, make_styles(), build_pdf() 골격 작성
- 표지(제목, 연도) 추가
- ★★★ 최빈출 첫 번째 과목 항목 작성 (섹션 C에서 분류한 3~4년 출제 유형)

> **멈춤 포인트:** D-1 완료. 작성한 내용을 보고하고 D-2로 넘어갈지 확인을 받는다.

#### D-2: ★★★ 최빈출 두 번째 과목 추가

**작업 내용:**
- 기존 스크립트에 ★★★ 최빈출 두 번째 과목 항목 추가

> **멈춤 포인트:** D-2 완료. 확인 후 D-3a로 넘어간다.

#### D-3a: ★★ 준빈출 표 추가

**작업 내용:**
- ★★ 준빈출 항목 표(2년 출제) 추가
- 스크립트 실행하여 오류 없는지 확인

> **멈춤 포인트:** D-3a 완료. 확인 후 D-3b로 넘어간다.

#### D-3b: 주제별 핵심 정리 추가

**작업 내용:**
- 주제별 핵심 포인트 요약 표 추가 (감별표, 배지·염색표, 면역세포표 등)
- 스크립트 실행하여 오류 없는지 확인

> **멈춤 포인트:** D-3b 완료. 확인 후 D-4로 넘어간다.

#### D-4: 연도별 출제 현황 + 시험 전략 TIP + 실행

**작업 내용:**
- 연도별 출제 현황 매트릭스 표 추가
- 시험 전략 TIP 박스 추가
- 스크립트 실행하여 PDF 정상 생성 확인
- 오류 발생 시 수정 후 재실행

**완료 기준:** PDF 파일 정상 생성 확인

> **멈춤 포인트:** D-4 완료. 생성된 PDF 경로와 내용을 보고하고, 커밋/다음 교시 진행 여부를 확인받는다.

| 규칙 | 내용 |
|------|------|
| 교시 순서 | 1교시 → 2교시 → 3교시 (한 번에 하나씩) |
| 섹션 순서 | A → B → C → D (건너뛰지 않음) |

## 프로젝트 개요

- **목적:** KAMC(기초의학종합평가) 빈출 유형 분석 PDF 생성
- **대상 연도:** 2021, 2023, 2024, 2025
- **교시별 스크립트:**
  - `make_pdf.py` — 1교시 (해부학)
  - `make_pdf_2교시.py` — 2교시 (생화학)
  - `make_pdf_3교시.py` — 3교시 (생리학)
  - `make_pdf_4교시.py` — 4교시 (병리학 + 기생충학)
  - `make_pdf_5교시.py` — 5교시 (미생물학 + 면역학)
- **개발 브랜치:** `claude/fifth-period-session-3Oj7Q`
- **원격 저장소:** `p16791918-lab/gjp`
- **한글 폰트:** NanumGothic (`/usr/share/fonts/truetype/nanum/`) — 없으면 `apt-get install -y fonts-nanum` 먼저 실행
